# Install this bitch into a virtualenv or pyenv. 

## virtualenv - *NIX
```
python -m venv venv
source ./venv/bin/activate
pip install -r requirements.txt
```

## virtualenv - M$
```
python -m venv venv
.\venv\Scripts\activate.bat
pip.exe install -r requirements.txt
```
